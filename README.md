# rog_okoshi

Zoomなどの文字起こしテキストを、インタビュー記事向けの体裁に整える社内向けツールです。

## できること

- 話者つき文字起こしを貼り付けて整形
- インタビュアーの発言に `・` を付与
- 回答は改行で接続
- 事実説明中の軽いフィラーを一部削除
- 整形結果のコピーと `txt` 保存

## 使い方

1. `index.html` をブラウザで開く
2. インタビュアー名を入力する
3. Zoom文字起こしを貼り付ける
4. `整形する` を押す

インタビュアー名が未入力の場合は、最初の話者をインタビュアーとして扱います。

## 想定入力例

```text
山田: 今日はよろしくお願いします。
佐藤: よろしくお願いします。
山田: 今回の取り組みについて教えてください。
佐藤: えーと、社内記事の整形を効率化したいと考えています。
```
