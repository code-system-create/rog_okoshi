# インタビュー整形ツール 使い方メモ

## これは何ができる？

このツールは、Zoomの文字起こしを貼り付けると、インタビュー記事用の読みやすい形式に整えるためのものです。

- インタビュアーの発言の先頭に `・` を付ける
- 回答者の発言をその下に改行して並べる
- `WEBVTT`、連番、タイムスタンプを削除する
- インタビュアー側の `ありがとうございます` や一部のフィラーを削除する
- 不自然に残った句読点をある程度整える

## 使い方

1. ツールを開く
2. Zoomの文字起こしを、そのまま `文字起こしテキスト` に貼り付ける
3. `整形する` を押す
4. 整形結果を確認する
5. 問題なければ `コピー` または `txtで保存` を使う

## 前提

- `インタビュー担当` という話者名を、インタビュアーとして扱います
- Zoomの文字起こしに話者名が入っている前提です

## できること

- Zoomの文字起こしをそのまま貼り付けて使える
- `WEBVTT` 形式の余分な行を自動で外せる
- インタビュアー側の進行用の言葉を一部整理できる
- `ありがとうございます` を自動で削除できる
- 質問文らしい文末を `？` に寄せられる

## できないこと

- 音声ファイルそのものを直接読み込むこと
- 話者名がない文字起こしを正確に整理すること
- 認識ミスのある文字起こしを完全に正しく直すこと
- 文脈を100%理解して、自然な記事表現に完全変換すること
- `→` と `・` を自動で正確に使い分けること

## 注意点

- 回答者側のフィラーは基本的に残ります
- 文字起こしの誤変換はそのまま残ることがあります
- インタビュアー側でも、言い回しによっては不自然な文が残ることがあります
- 相づちや前置きを削除した結果、まれに文のつながりが不自然になることがあります
- `インタビュー担当` 以外の名前がインタビュアーになっている場合は、そのままでは意図どおり整形されません

## 最後に自分で確認すること

- 話者の切り分けが合っているか
- 質問と回答の対応がずれていないか
- 削除された言葉で意味が変わっていないか
- 文字起こしの誤変換が残っていないか
- 句読点や改行が不自然でないか
- 社外共有できる表現になっているか

## よくある修正ポイント

- `56。になります。` のような不自然な表現を直す
- 誤変換された固有名詞や専門用語を直す
- 長すぎる質問文を読みやすく整える
- 回答の言いよどみを、必要に応じて手で調整する
